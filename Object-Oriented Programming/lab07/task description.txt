Клас Array – одновимірний масив. Пам’ять під елементи масиву повинна виділятися динамічно.
1) Знаходження максимального значення.                                                                                           +
2) Знаходження мінімального значення.                                                                                            +
3) Знаходження середнього арифметичного значення масиву.                                                                         +
4) Сортування елементів масиву методом вибірки за спаданням.                                                                     +
5) Сортування елементів масиву методом бульбашки за зростанням.                                                                  +
6) Зміна розмірів масиву.                                                                                                        +
7) Додавання (почленне додавання елементів масиву)                                                                               +
8) Віднімання (почленне віднімання елементів масиву)                                                                             +
9) Множення на скаляр.                                                                                                           +
10) Введення масиву з StringGrid (>>)                                                                                            +
11) Виведення масиву у StringGrid (<<)                                                                                           +
12) Введення масиву з ListBox (>>)                                                                                           неможливо
13) Виведення масиву у ListBox (<<)                                                                                              +
14) Виведення масиву у Memo (<<)                                                                                                 +
15) Забезпечити можливість отримання значення елементу [i] подібно до доступу до елементів звичайного одновимірного масиву.      +
16) Розробити для класу конструктор за замовчуванням та декілька звичайних конструкторів.                                        +
17) Створити конструктор копіювання.                                                                                             +
18) Перевантажити операцію присвоєння.                                                                                           +
19) Створити деструктор для вивільнення динамічно виділеної пам’яті.                                                             +
20) Об'єкти класу розмістити в динамічній пам'яті.                                                                               +
21) Продемонструвати розроблені можливості класу завдяки створеному віконному застосуванню.                                      +
